#### 2.1 Mamba模块设计方案

**新建文件：** `modules/mamba_temporal.py`

**核心设计：**
```python
class MambaTemporal(nn.Module):
    """
    替代原RNN类的Mamba时序建模模块
    
    优势：
    1. 线性复杂度O(L) vs GRU的O(L*h)
    2. 选择性记忆机制，自适应保留关键信息
    3. 长距离依赖建模能力更强
    """
    def __init__(
        self,
        n_input=1408,      # PointNet2(1024) + PoinTr(384)
        n_output=72,       # 24关节 × 3坐标
        d_model=1024,      # Mamba隐藏维度
        d_state=16,        # SSM状态维度
        d_conv=4,          # 局部卷积核大小
        expand=2,          # 扩展因子
        n_layers=2,        # Mamba层数
        dropout=0.1
    ):
        super().__init__()
        
        # 输入投影：1408 → 1024
        self.input_proj = nn.Linear(n_input, d_model)
        
        # Mamba层堆叠（带残差）
        self.mamba_layers = nn.ModuleList([
            Mamba(d_model=d_model, d_state=d_state, d_conv=d_conv, expand=expand)
            for _ in range(n_layers)
        ])
        self.layer_norms = nn.ModuleList([
            nn.LayerNorm(d_model) for _ in range(n_layers)
        ])
        
        # 输出投影：1024 → 72
        self.output_proj = nn.Linear(d_model, n_output)
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x):
        """
        Args:
            x: (B, T, 1408) 时序特征序列
        Returns:
            out: (B, T, 72) 预测的关节坐标
        """
        # 输入投影
        x = F.gelu(self.input_proj(x))
        
        # Mamba层处理（带残差连接）
        for mamba, norm in zip(self.mamba_layers, self.layer_norms):
            residual = x
            x = norm(x)
            x = mamba(x)
            x = x + residual  # 残差连接
        
        # 输出投影
        out = self.output_proj(self.dropout(x))
        return out
```

#### 2.2 时序一致性损失设计

**修改文件：** `modules/loss.py`

**新增类：** `TemporalConsistencyLoss`

**核心设计：**
```python
class TemporalConsistencyLoss(nn.Module):
    """
    时序一致性损失函数
    
    包含三个约束项：
    1. 速度一致性：||v_t - v_{t-1}||²，鼓励平滑运动
    2. 加速度一致性：||a_t - a_{t-1}||²，减少抖动
    3. 骨骼长度一致性：Var(bone_length_t)，保持骨架稳定
    """
    def __init__(
        self,
        velocity_weight=1.0,
        acceleration_weight=0.5,
        bone_length_weight=0.3
    ):
        super().__init__()
        self.velocity_weight = velocity_weight
        self.acceleration_weight = acceleration_weight
        self.bone_length_weight = bone_length_weight
        
        # SMPL 24关节骨架连接
        self.skeleton_pairs = [
            (0, 1), (0, 2), (0, 3),  # 脊柱
            (1, 4), (2, 5), (3, 6),  # 肩膀
            (4, 7), (5, 8), (6, 9),  # 肘部
            (7, 10), (8, 11), (9, 12),  # 手腕
            (0, 13), (0, 16),  # 髋部
            (13, 14), (14, 15),  # 左腿
            (16, 17), (17, 18),  # 右腿
            (15, 19), (15, 20), (18, 21), (18, 22)  # 脚踝
        ]
    
    def forward(self, joints):
        """
        Args:
            joints: (B, T, 24, 3) 预测关节序列
        Returns:
            dict: 各项损失值
        """
        loss_dict = {}
        
        # 速度损失
        if joints.size(1) >= 2:
            velocity = joints[:, 1:] - joints[:, :-1]
            if velocity.size(1) >= 2:
                velocity_diff = velocity[:, 1:] - velocity[:, :-1]
                loss_dict['loss_velocity'] = torch.mean(velocity_diff ** 2) * self.velocity_weight
        
        # 加速度损失
        if joints.size(1) >= 3:
            velocity = joints[:, 1:] - joints[:, :-1]
            acceleration = velocity[:, 1:] - velocity[:, :-1]
            if acceleration.size(1) >= 2:
                accel_diff = acceleration[:, 1:] - acceleration[:, :-1]
                loss_dict['loss_acceleration'] = torch.mean(accel_diff ** 2) * self.acceleration_weight
        
        # 骨长一致性损失
        if joints.size(1) >= 2:
            bone_loss = 0
            for ja, jb in self.skeleton_pairs:
                bone_lengths = torch.norm(
                    joints[:, :, jb] - joints[:, :, ja], dim=-1
                )  # (B, T)
                bone_loss += torch.var(bone_lengths, dim=1).mean()
            loss_dict['loss_bone'] = bone_loss * self.bone_length_weight
        
        # 总损失
        total = sum(loss_dict.values())
        loss_dict['loss_temporal'] = total
        
        return loss_dict
```

**修改位置：** `modules/loss.py:15-92` 的 `Loss` 类

**修改内容：**
```python
class Loss(nn.Module):
    def __init__(self):
        super().__init__()
        self.criterion_param = nn.MSELoss()
        self.criterion_joints = nn.MSELoss()
        self.criterion_vertices = nn.MSELoss()
        self.chamfer_loss = ChamferDistanceL1()
        self.smpl = SMPL()
        
        # 新增：时序一致性损失
        self.temporal_loss = TemporalConsistencyLoss()
        self.temporal_weight = 0.1  # 可配置
    
    def forward(self, **kw):
        # ... 现有损失计算 ...
        
        # 新增：时序一致性损失
        if 'pred_full_joints' in kw:
            pred_full_joints = kw['pred_full_joints']
            temporal_dict = self.temporal_loss(pred_full_joints)
            details.update(temporal_dict)
        
        # ... 损失累加 ...
        loss = 0
        for k, v in details.items():
            if k == 'loss_temporal':
                loss += self.temporal_weight * v
            else:
                loss += v
        
        details['loss'] = loss
        return loss, details
```

## 📚 创新点学术论证

### 创新点1：Mamba在LiDAR姿态估计中的首次应用

**理论依据：**
- Mamba的状态空间模型具有线性复杂度O(L)，适合处理长序列
- 选择性状态机制可自适应保留关键运动特征
- 相比GRU，具有更强的长距离依赖建模能力

**与现有工作对比：**
| 方法 | 时序模型 | 复杂度 | 长距离依赖 |
|------|---------|--------|-----------|
| DAPT | 无（单帧） | - | ❌ |
| LiDARCap (原) | GRU | O(L×h) | 一般 |
| Transformer方法 | Attention | O(L²) | ✅ |
| **Ours** | Mamba | O(L) | ✅ |

### 创新点2：多约束时序一致性损失

**理论依据：**
- 速度一致性约束一阶导数平滑
- 加速度一致性约束二阶导数平滑
- 骨长一致性约束物理合理性

**数学表达：**
```
L_temporal = λ_v * ||v_t - v_{t-1}||² 
           + λ_a * ||a_t - a_{t-1}||²
           + λ_b * Var(bone_length)
```

### 创新点3：预留GCSP多模态融合架构

**设计理念（参考STM-Fuse）：**
- 门控跨模态状态传播
- 动态学习模态可靠性
- 自适应信息融合

**预留意义：**
- 为后续研究提供清晰路径
- 架构设计合理，易于扩展
- 保持代码结构完整性
