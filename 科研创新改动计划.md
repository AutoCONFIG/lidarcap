## 研究主题

**ST-Mamba-LiDARCap: Spatio-Temporal Selective Mamba with RGB-LiDAR Fusion for Human Pose Estimation**

基于时空选择性Mamba与RGB-LiDAR双模态融合的人体姿态估计

---

## 核心定位

| 项目 | 说明 |
|------|------|
| **任务** | 点云补全 + 姿态重建（非预测） |
| **核心问题** | 如何自适应地融合时序信息与多模态信息来提升重建精度？ |
| **数据集** | LiDAR点云 + RGB图像（动态跟踪裁剪，人物模糊，背景移动） |

**数据集限制与应对**:

| 限制 | 应对策略 |
|------|----------|
| 无姿态分类标注 | 不需要，基于点云质量自适应 |
| 无运动类型标注 | 自动计算帧间差异判断运动速度 |
| 无遮挡/可见性标注 | 自动计算局部点云数量判断可见性 |
| 无相机内参/外参 | 自适应投影角度搜索 |
| RGB人物模糊 | YOLO分割 + 双mask融合 |

### 第二阶段: 多模态预处理 ⏳ 待开发

**目标**: 实现双模态mask融合，解决RGB背景干扰问题

#### 模块3+4: 双模态mask融合 + 自适应投影角度搜索

**问题**:
- RGB图像背景移动，人物模糊
- 无相机内参/外参，无法直接投影

**方案**:
```
预处理阶段 (离线):
┌─────────────────────────────────────────────────────────┐
│  RGB序列 ──→ YOLO分割 ──→ YOLO_mask序列                 │
│                                │                        │
│                                ↓                        │
│  点云序列 ──→ 投影角度搜索 ──→ 最优(θ, φ)               │
│              (最大化IoU)         │                      │
│                                ↓                        │
│          点云投影(θ, φ) ──→ 点云_mask序列               │
│                                │                        │
│                                ↓                        │
│              融合mask = YOLO_mask ∩ 点云_mask           │
│                                │                        │
│                                ↓                        │
│                      存入HDF5供训练使用                  │
└─────────────────────────────────────────────────────────┘
```

**自适应投影角度搜索算法**:
```python
def find_best_projection_angle(point_cloud, yolo_mask):
    """搜索与YOLO mask吻合度最高的投影角度"""
    best_iou = 0
    best_params = (0, 0)
    
    for theta in range(-90, 91, 5):      # 俯仰角
        for phi in range(0, 360, 10):    # 偏航角
            proj_mask = orthographic_projection(point_cloud, theta, phi)
            iou = compute_iou(proj_mask, yolo_mask)
            if iou > best_iou:
                best_iou = iou
                best_params = (theta, phi)
    
    return best_params
```

**双mask融合优势**:
| mask来源 | 优势 |
|----------|------|
| YOLO_mask | 精确边缘 |
| 点云_mask | 3D一致性校验 |
| 融合mask | 交集过滤误检，信息量最大 |

**新增文件**: `scripts/preprocess_rgb_mask.py`

### 第五阶段: 不确定性估计 ⏳ 待开发

**目标**: 为每个关节预测估计置信度

#### 模块7: 不确定性估计

```python
# 输出
pose = (B, T, 72)           # 关节坐标
uncertainty = (B, T, 24)    # 每个关节的不确定性
```

**损失函数**: `L = exp(-u) * ||pose - gt||^2 + u`

**协同效应**: 不确定性 → 指导选择性注意力权重

## 开发原则

1. **渐进式开发**: 每个阶段独立可测试，便于消融实验
2. **模块化设计**: 各创新点独立模块，可单独开关
3. **配置驱动**: 所有超参数通过配置文件控制
4. **自动信号优先**: 不依赖额外标注，基于可计算信号
5. **双模态互补**: RGB提供精确边缘，点云提供3D一致性
