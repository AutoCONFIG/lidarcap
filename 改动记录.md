##### 修改前（单向Mamba）
```python
class MambaTemporal(nn.Module):
    def __init__(self, n_input, n_output, n_hidden, d_state=16, d_conv=4, expand=2):
        """
        Mamba-based temporal fusion module
        
        Args:
            n_input: Input feature dimension (e.g., 1024 + 384 = 1408)
            n_output: Output dimension (e.g., 24 * 3 = 72)
            n_hidden: Hidden dimension for the Mamba model
            d_state: SSM state expansion factor
            d_conv: Local convolution width
            expand: Block expansion factor
        """
        super().__init__()
        self.n_hidden = n_hidden
        
        self.linear1 = nn.Linear(n_input, n_hidden)
        self.dropout = nn.Dropout()
        
        try:
            from mamba_ssm import Mamba
            self.mamba = Mamba(
                d_model=n_hidden,
                d_state=d_state,
                d_conv=d_conv,
                expand=expand,
                batch_first=True
            )
        except ImportError:
            raise ImportError(
                "mamba_ssm is not installed. Install with: pip install mamba-ssm"
            )
        
        self.linear2 = nn.Linear(n_hidden, n_output)
        
    def forward(self, x):
        """
        Args:
            x: Input tensor of shape (B, T, D)
        
        Returns:
            Output tensor of shape (aB, T, n_output)
        """
        x = F.relu(self.dropout(self.linear1(x)), inplace=True)
        
        x = self.mamba(x)
        
        x = self.linear2(x)
        
        return x
```

##### 修改后（双向Mamba）
```python
class MambaTemporal(nn.Module):
    def __init__(self, n_input, n_output, n_hidden, d_state=16, d_conv=4, expand=2):
        """
        Mamba-based temporal fusion module (Bidirectional)
        
        Args:
            n_input: Input feature dimension (e.g., 1024 + 384 = 1408)
            n_output: Output dimension (e.g., 24 * 3 = 72)
            n_hidden: Hidden dimension for the Mamba model
            d_state: SSM state expansion factor
            d_conv: Local convolution width
            expand: Block expansion factor
        """
        super().__init__()
        self.n_hidden = n_hidden
        
        self.linear1 = nn.Linear(n_input, n_hidden)
        self.dropout = nn.Dropout()
        
        try:
            from mamba_ssm import Mamba
            # 前向Mamba
            self.mamba_fwd = Mamba(
                d_model=n_hidden,
                d_state=d_state,
                d_conv=d_conv,
                expand=expand,
                batch_first=True
            )
            # 后向Mamba
            self.mamba_bwd = Mamba(
                d_model=n_hidden,
                d_state=d_state,
                d_conv=d_conv,
                expand=expand,
                batch_first=True
            )
        except ImportError:
            raise ImportError(
                "mamba_ssm is not installed. Install with: pip install mamba-ssm"
            )
        
        # 双向输出拼接，所以是 n_hidden * 2
        self.linear2 = nn.Linear(n_hidden * 2, n_output)
        
    def forward(self, x):
        """
        Args:
            x: Input tensor of shape (B, T, D)
        
        Returns:
            Output tensor of shape (B, T, n_output)
        """
        x = F.relu(self(self.dropout(self.linear1(x)), inplace=True)
        
        # 前向处理
        x_fwd = self.mamba_fwd(x)
        
        # 后向处理 - 反转时间维度
        x_bwd = self.mamba_bwd(x.flip(1)).flip(1)
        
        # 拼接双向输出
        x = torch.cat([x_fwd, x_bwd], dim=-1)
        
        x = self.linear2(x)
        
        return x
```

##### 核心改动说明

1. **双向处理机制**:
   - 从单向Mamba改为双向Mamba
   - 与原 [`RNN`](modules/regressor.py:226) 类的双向GRU（`bidirectional=True`）保持完全等价

2. **维度匹配**:
   - 输出层从 `nn.Linear(n_hidden, n_output)` 改为 `nn.Linear(n_hidden * 2, n_output)`
   - 这是因为双向输出拼接后维度翻倍

3. **前向传播逻辑**:
   ```python
   # 前向处理 (t=0 → t=T)
   x_fwd = self.mamba_fwd(x)
   
   # 后向处理 (t=T → t=0)，通过时间反转实现
   x_bwd = self.mamba_bwd(x.flip(1)).flip(1)
   
   # 拼接双向输出
   x = torch.cat([x_fwd, x_bwd], dim=-1)
   ```

4. **数据流对比**:

   ```mermaid
   flowchart LR
       subgraph 原双向GRU
           A["(B,T,1408)"] --> B["Linear→1024"] --> C["双向GRU<br/>bidirectional=True"] --> D["Linear 2048→72"] --> E["(B,T,72)"]
       end
       subgraph 改进后双向Mamba
           A2["(B,T,1408)"] --> B2["Linear→1024"] 
           B2 --> C2["前向Mamba<br/>t=0→T"]
           B2 --> C3["后向Mamba<br/>t=T→0 (flip)"]
           C2 --> D2["cat<br/>dim=-1"]
           C3 --> D2
           D2 --> E2["Linear 2048→72"] --> F2["(B,T,72)"]
       end
   ```

### 改动原因

#### 问题分析
原 [`RNN`](modules/regressor.py:226) 类使用的是**双向GRU**：

```python
# modules/regressor.py:226-233
class RNN(nn.Module):
    def __init__(self, n_input, n_output, n_hidden, n_rnn_layer=2):
        super(RNN, self).__init__()
        self.rnn = nn.GRU(n_hidden, n_hidden, n_rnn_layer,
                          batch_first=True, bidirectional=True)  # 双向
        self.linear1 = nn.Linear(n_input, n_hidden)
        
        self.linear2 = nn.Linear(n_hidden * 2, n_output)  # *2 因为双向
```

为什么需要双向？
- **前向传播**：利用过去时刻的信息预测当前状态
- **后向传播**：利用未来时刻的信息增强当前状态的表示
- 时序建模中，未来信息对当前时刻的预测也很重要（例如手势识别）

#### 潜在问题
如果使用单向Mamba：
- ❌ 丢失了后向时序依赖信息的捕获能力
- ❌ 输出维度不匹配（1024 vs 2048），可能导致模型性能下降
- ❌ 与原设计不一致，无法公平对比

### 改动技术细节

#### Mamba模块参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `d_model` | `n_hidden` (1024) | Mamba模型维度 |
| `d_state` | 16 | SSM状态扩张因子 |
| `d_conv` | 4 | 局部卷积核宽度 |
| `expand` | 2 | 块扩张因子 |
| `batch_first` | True | 输入格式为 (B, T, D) |

#### 计算复杂度对比

| 模型 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 双向GRU | O(T × n_hidden²) | O(n_hidden²) |
| 双向Mamba | O(T × n_hidden) | O(n_hidden) |
| Transformer | O(T² × n) | O(T² + n) |

**优势**：Mamba在长序列上具有线性复杂度优势
